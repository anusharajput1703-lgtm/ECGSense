# 🫀 ECGSense

## AI-Assisted Arrhythmia Detection using Deep Learning

ECGSense is a Streamlit-based web application developed for automatic ECG heartbeat classification using a Convolutional Neural Network (CNN).

## Features

- Predicts one of five heartbeat classes
- Uses a trained CNN model
- Interactive ECG waveform visualization
- Displays prediction confidence
- Heartbeat class information
- Dataset overview
- CNN architecture explanation
- Modern clinical-style dashboard

## Dataset

- MIT-BIH Arrhythmia Dataset
- Training Samples: 87,554
- Testing Samples: 21,892
- Signal Length: 187 samples

## Technologies Used

- Python
- TensorFlow / Keras
- Streamlit
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Heartbeat Classes

- N – Normal Beat
- S – Supraventricular Beat
- V – Ventricular Beat
- F – Fusion Beat
- Q – Unknown Beat

## Model

- Deep Learning Model: Convolutional Neural Network (CNN)
- Input Length: 187
- Output Classes: 5

This project is intended for educational and research purposes only and should not be used for clinical diagnosis.
